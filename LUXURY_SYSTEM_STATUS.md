## 🚀 Using Your New Components

### Basic Usage
```typescript
import { Button, Input, Card, Badge } from '@/shared/components';

// Luxury Gold Button
<Button variant="primary" size="lg">Get Started</Button>

// Neon Glass Input
<Input label="Email" placeholder="Enter email" />

// Premium Card
<Card variant="glass" hoverable>Premium Content</Card>

// Status Badge
<Badge variant="success">ACTIVE</Badge>
```

### All Component Variants

**Button**: primary (gold) | secondary (cyan) | danger (fire) | ghost  
**Card**: default (glass) | glass (premium) | elevated (neumorphic) | hoverable  
**Badge**: default | success (green) | warning (gold) | danger (red) | info (cyan)  
**Input**: All inputs now have glass background + neon glow
