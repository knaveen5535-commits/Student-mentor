# API Features

## Overview
Each feature folder contains all logic needed for that feature:
- `controller.js` - Express request handlers
- `routes.js` - Route definitions  
- `service.js` - Business logic layer
- `types.js` - TypeScript types
- `README.md` - Feature documentation

## Features

### Auth
- Google OAuth 2.0 setup
- JWT token management
- Session handling
- User registration & login

### Chat
- AI conversation via OpenRouter
- Message history
- Conversation management
- Streaming responses

### Projects
- Create & manage projects
- Project search & filtering
- Project history
- Collaboration features

### Tutorials
- YouTube search integration
- Save favorite tutorials
- Tutorial recommendations
- Learning paths

### Profile
- User profile management
- Settings & preferences
- Stats & achievements
- Account management

### Upload
- File upload handling
- Image processing
- Document storage
- Camera capture processing
